
# setup

**install Xcode from App Store**

**install Xcode's command line tools**
```
xcode-select -—install
```

**install homebrew**
```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew doctor
```

**install application from Brewfile**
```
brew bundle
```
